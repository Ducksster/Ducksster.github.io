How I learned programming.

For as long as I can remember I've always been interested in making video games. When

I was very little I used to believe that games were made by saving an absurd amount of

screenshots and strategically switching between them for each input. Later I learned

that games were made with programming but I believed that was far too complicated for

me, so instead I set out to find a way to make games without coding. Eventually, I

found a website called Sploder. Sploder allowed its users to make games without code.

I excitedly made games and levels twice or more a week, but I thought it was very

limiting. Eventually, I decided to learn programming with Roblox. Roblox games are

made with Lua, so I had to learn the language. I immediately loved it so I practiced

every day by watching videos and making things myself. I used Roblox and Lua for

everything from games to visualizing functions. Now, I've learned many more languages

and platforms that I use for different things.
